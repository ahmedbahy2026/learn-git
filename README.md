## Binary Search
```cpp
int l=0,r=INF,ans=-1;
while(l<=r){
    int mid = (l+r)/2;
    if(ok(mid)){
	ans = mid;
	r=mid-1;
    }else l=mid+1;
}
cout<<ans<<"\n";
```
